Twailer

Get an email whenever a tweet with a certain hashtag or user mention is published.

View project onGitHub

How does it work?

Subscribe

If you want to get email updates for a hashtag or user mention, simply send an email to subscribe@twailer.mstaessen.be with the hashtag (including #) or user mention (including @) in the subject field. If the subscription was successful, you will receive a confirmation email.

Unsubscribe

If you no longer want to receive email updates, you can unsubscribe in a similar fasion: send an email to unsubscribe@twailer.mstaessen.be. You can leave the subject line empty if you want to cancel all of your subscriptions or you can specify which hashtag or user mention you want to unfollow.

Self Hosted

You can install twailer on your own mail server. For more information on how to do that, take a look at the README file.
